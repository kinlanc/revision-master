📚 Revision Master

Revision Master is an interactive, browser-based study planner designed to help learners visualise their progress and master their curriculum.

Unlike traditional planners, this tool is privacy-first. It requires no login, no database, and no tracking. All data is stored locally in the learner's browser.

✨ Key Features

Traffic Light Tracking: Categorise topics by confidence (Focus, Building, Mastered).

Session Logging: Track the last time you reviewed a specific topic to beat the forgetting curve.

Curriculum Agnostic: Completely customizable for any subject or exam board (IB, IGCSE, A-Levels, etc.).

Portable App Export: Learners can download their specific planner as a self-contained HTML file to keep their data forever.

Print Optimised: Generate clean, professional PDF reports of study progress.
